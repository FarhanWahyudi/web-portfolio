B-University

LARAVEL

MYSQL

TAILWIND

Description

Project ini saya kerjakan sebagai bagian dari pembelajaran di kelas Laravel yang diselenggarakan oleh Build With Angga. Aplikasi ini dirancang sebagai sistem manajemen universitas, yang memungkinkan admin untuk mengelola berbagai data seperti berita, pengumuman, dosen, dan mahasiswa melalui antarmuka yang bersih, modern, dan intuitif.

Pengembangan project ini tidak hanya mengasah kemampuan saya dalam menggunakan Laravel sebagai framework backend, tetapi juga mengajarkan praktik pengembangan aplikasi web yang rapi, terstruktur, dan efisien. Dengan memanfaatkan Filament, saya dapat membangun dashboard admin yang dinamis dan powerful dengan sangat cepat. Desain tampilan dibuat dengan Tailwind CSS untuk memastikan bahwa website tampil responsif di berbagai perangkat. Sementara itu, seluruh data disimpan dan dikelola menggunakan MySQL sebagai basis data.

Project ini mengintegrasikan banyak konsep penting dalam pengembangan web seperti autentikasi pengguna, manajemen data relational, validasi input, hingga optimalisasi UI/UX menggunakan komponen Tailwind dan sistem layout yang fleksibel. Saya juga terbiasa bekerja dengan struktur MVC Laravel serta memanfaatkan fitur-fitur modern seperti route resource, Eloquent ORM, dan Livewire.

Sistem Gudang

LARAVEL

MYSQL

TAILWIND

Description

Setelah menyelesaikan pembelajaran Laravel, saya mengembangkan project pribadi berupa Aplikasi Sistem Gudang yang berfungsi untuk mencatat dan memantau pergerakan stok barang dalam sebuah lingkungan penyimpanan atau inventaris. Aplikasi ini dirancang untuk memudahkan admin dalam mengelola data barang, mencatat stok masuk dan keluar, serta menampilkan laporan ringkas per minggu langsung dari halaman dashboard.

Dengan menggunakan Laravel sebagai backend utama, saya membangun struktur aplikasi yang terorganisir dan efisien. Sistem ini memanfaatkan Tailwind CSS untuk antarmuka pengguna yang bersih dan responsif, serta MySQL sebagai basis data untuk mencatat semua pergerakan stok secara akurat.

Dashboard admin dirancang agar informatif dan mudah dipahami, menampilkan grafik atau ringkasan data stok masuk dan keluar dalam 7 hari terakhir. Admin dapat menambahkan produk baru, mengedit informasi produk yang ada, serta mencatat transaksi masuk dan keluar barang. Seluruh fitur dilengkapi dengan validasi input dan notifikasi yang jelas, sehingga meminimalkan kesalahan input data.
